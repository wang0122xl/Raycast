Raycast 扩展和脚本命令的集合，用于开发者工作流自动化。

## 项目结构

```
extensions/          # Raycast 扩展（每个子目录是一个独立扩展）
scripts/             # Raycast 脚本命令（独立 shell 脚本）
```
